# Placeholder Files

To make the website functional, you need to add:

## In the `gifs/` folder:
- sample1.gif (animated GIF)
- sample1.png (static version - first frame of the GIF)
- sample2.gif
- sample2.png
- sample3.gif
- sample3.png
- sample4.gif
- sample4.png

## In the `audio/` folder:
- sample1.mp3
- sample2.mp3
- sample3.mp3
- sample4.mp3

## Quick Start with Placeholder Content

You can create simple placeholders to test:

1. **For static images (.png files)**: Use any image file
2. **For GIFs**: Use any animated GIF
3. **For audio**: Use any MP3 or WAV file

Then update the `samplerConfig` in `script.js` to match your actual filenames.

## Example Workflow:

1. Create/find 4 GIFs you want to use
2. Export the first frame of each as a PNG
3. Create/find 4 audio samples (loops work great!)
4. Place them in the respective folders
5. Update `script.js` with the exact filenames
6. Open `index.html` in your browser

The website will work as soon as these files are in place!
